# Metabolomics metadata

As part of the [X-omics initiative](https://www.x-omics.nl/), we aim to design a sample data schema that allows for integration of metabolomics data with other types of omics data. This effort will be in alignment with existing standardization initiatives ([metabolomics-elixir][1]).
